# Eenvoudige Klantbeheer Applicatie

Ontwikkel een eenvoudige consoleapplicatie in C# voor het beheren van klanten. De applicatie moet de mogelijkheid bieden om klanten toe te voegen, te bekijken en te verwijderen. Gebruik de principes van objectgeoriënteerd programmeren en zorg voor een duidelijke structuur. De focus ligt op het correct implementeren van klassen en methoden om de functionaliteit te realiseren.

## Opdracht

Implementeer de gevraagde functionaliteit.
